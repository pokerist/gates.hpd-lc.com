# بوابة التحقق من بطاقة الرقم القومي (MVP)

واجهة عربية مبسطة للأمن والأدمن مع Manual Debug لاستخراج البيانات من بطاقة الرقم القومي.

## المزايا
- واجهة موبايل أولًا للأمن مع كاميرا وفريم مناسب للبطاقة.
- استخراج الاسم الكامل والرقم القومي (14 رقم) باستخدام YOLO + EasyOCR.
- Tesseract كـ fallback ولوج مفصل في الكونسول.
- قاعدة بيانات SQLite لإدارة الدخول والحظر.
- لوحة أدمن للبحث والحظر والحذف.

## التشغيل على Ubuntu
```
cd id_gate_mvp
bash deploy.sh
```
سيتم التشغيل على `http://<server-ip>:5000`.

## المسارات
- `/` الصفحة الرئيسية
- `/security` واجهة الأمن
- `/admin` لوحة الأدمن
- `/debug` Manual Debug

## ملاحظات
- النماذج موجودة داخل `models/`.
- يتم حفظ قاعدة البيانات في `data/gate.db`.
- يتم حفظ صور الـ Debug في `data/debug/`.
